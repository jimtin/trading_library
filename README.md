# Python Trading Library
Multi-exchange trading library platform for Python. 

## Supported Platforms
- [MetaTrader 5](https://www.metatrader5.com/en)

## Functions
### Simple Moving Average (SMA)
- 50-day SMA
- 200-day SMA

### Cross Events
- Golden Cross
- Death Cross
